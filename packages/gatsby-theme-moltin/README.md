## Install

```sh

```

## How to use

```js
// In your gatsby-config.js
plugins: [
  {
    resolve: `@moltin/gatsby-theme-moltin`,
    options: {
      client_id: '...',
      basePath: '/', // basePath: '/store'
    },
  },
]
```
